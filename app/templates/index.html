Pipeline

Four steps.
One clean output.

01
Fetch
Enter an ArXiv ID, URL, or keyword. The app queries the ArXiv API and retrieves the paper's metadata and abstract.
02
Clean
LaTeX commands, math expressions, and unicode artifacts are stripped. The text is chunked into token-aware windows.
03
Summarize
LangChain routes the text through LLaMA 3 on Groq. Long papers use Map-Reduce. Short ones use a single structured prompt.
04
Explain
Technical jargon is auto-detected and explained in plain English with real-world analogies — no PhD required.
